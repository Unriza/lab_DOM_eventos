Contexto del Laboratorio
En este laboratorio construirás desde cero una mini tienda en línea con carrito de compras funcional. Usarás
HTML para la estructura, Bootstrap para el diseño visual y JavaScript para hacer que todo cobre vida al
interactuar con el DOM.
Al finalizar tendrás una página donde el usuario puede agregar productos al carrito, eliminarlos individualmente,
ver el total actualizado y vaciar el carrito — todo sin recargar la página.
Herramientas de JavaScript que vas a usar
· querySelector / querySelectorAll — Seleccionar elementos del DOM
· addEventListener('click', ...) — Detectar cuando el usuario hace clic
· createElement / appendChild — Crear nuevos elementos y agregarlos al DOM
· textContent — Cambiar el texto visible de un elemento
· element.remove() — Eliminar un elemento del DOM
· dataset — Leer datos guardados en atributos data- del HTML
· forEach — Recorrer una lista de elementos y hacer algo con cada uno
